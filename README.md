# Send.Token

A minimal Next.js dApp for bulk-sending ERC-20 or native tokens on Lens chain.

---

## Quick start

```bash
git clone https://github.com/kkpsiren/senddottoken
cd senddottoken
npm i            # or npm install
cp .env.example .env  # fill in keys if needed
npm run dev
```
